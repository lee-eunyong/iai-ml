2. Pandas: 데이터 삭제 및 결측치 처리
데이터프레임(DataFrame)에서 불필요한 데이터를 제거할 때 사용하는 핵심 함수 두 가지입니다.

① dropna 함수 (결측치 제거)
데이터프레임 내에 값이 비어있는 **결측치(NaN)**가 포함된 행이나 열을 찾아 제거합니다.

df.dropna(): 기본적으로 어떤 행에든 하나 이상의 결측치가 있으면 해당 행을 제거합니다.

df.dropna(axis=1): 어떤 열에든 하나 이상의 결측치가 있으면 해당 열을 제거합니다.

df.dropna(subset=['컬럼명']): 특정 열(예: '컬럼명')에 결측치가 있는 행만 선택적으로 제거합니다.

② drop 함수 (명시적 삭제)
결측치 여부와 상관없이 사용자가 특정 행이나 열을 명시적으로 지정하여 삭제합니다.

df.drop(['컬럼명'], axis=1): 지정된 열을 제거합니다.

df.drop([index], axis=0): 지정된 행(인덱스 기준)을 제거합니다.

💡 요약: 결측치 처리에 집중할 때는 dropna를 사용하고, 데이터의 구조를 조정하거나 불필요한 부분을 제거할 때는 drop을 사용합니다.

3. Scikit-learn: 데이터 변환 메커니즘
머신러닝 모델을 학습시키기 전, 데이터를 가공하는 단계에서 공통적으로 사용되는 개념입니다.

① fit (학습)
데이터의 **특성(평균, 표준편차 등)**을 계산하고 학습하는 단계입니다.

예: 데이터의 평균이 50이고 표준편차가 10이라는 통계적 수치를 계산함.

② transform (변환)
fit 단계에서 계산된 통계 값을 이용하여 실제 데이터를 변환하는 단계입니다.

예: 학습된 평균과 표준편차를 바탕으로 데이터를 표준화(Standardization) 또는 정규화(Normalization) 수행함.
